# Contributing to Fortress

Thanks for your interest in contributing! 🎉

## How to Contribute

### 🐛 Report Bugs
Open an [issue](https://github.com/hyson810/fortress/issues) with:
- Steps to reproduce
- Expected vs actual behavior
- Environment details (OS, Go version, config)

### 💡 Suggest Features
Open a [discussion](https://github.com/hyson810/fortress/discussions) or [feature request](https://github.com/hyson810/fortress/issues/new?template=feature_request.md).

### 🛠️ Submit Code

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-idea`
3. Write tests for your changes
4. Run tests: `go test ./...`
5. Commit using [conventional commits](https://www.conventionalcommits.org/):
   - `feat:` new feature
   - `fix:` bug fix
   - `test:` tests
   - `docs:` documentation
   - `refactor:` code restructuring
   - `perf:` performance improvement
   - `chore:` maintenance
6. Push and open a Pull Request

### ✅ PR Checklist
- [ ] Code compiles (`go build ./...`)
- [ ] Tests pass (`go test ./...`)
- [ ] No new linter warnings
- [ ] Updated docs if needed
- [ ] Added tests for new functionality

### 🔬 Code Review
All submissions require review. Please be responsive to feedback.
